A blogging platform built with the **MERN stack** (MongoDB, Express, React, Node.js), with Redux Toolkit for state management and dark/light theme support.

Hosted Platforms
Frontend (Netlify): (https://your-frontend-site.netlify.app)
Backend (Render): [https://api-node-blogs.onrender.com]

client - Frontend Folder - cd client - npm install - npm run dev 
server - Backend Folder - cd server - npm install - npm start


It includes full user authentication (register, login, logout) using JWT tokens, blog CRUD operations (create, read, update, delete),
and blog pagination on the frontend. The application is deployed with the backend hosted on Render and the frontend on Netlify.
